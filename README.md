# rust wasm demo

## install required tools

```bash
cargo install wasm-pack
```


## build wasm

```bash
# wasm-pack build --target nodejs
# documentation: https://rustwasm.github.io/wasm-pack/book/commands/build.html#target
npm run build
```

## install dependencies for nodejs

```bash
npm install
```

## run

```bash
npm start
```
